This project is an AI-based Connect Four game developed using Python and Flask. The application allows users to play against a computer opponent powered by different AI strategies and intelligent move selection algorithms. It includes complete game state management, winner detection, move validation, and a responsive gameplay system.



Features 
🎮 Interactive Connect Four gameplay 
🤖 AI opponent with multiple strategies 
🔄 Real-time move validation and game updates 
🏆 Automatic winner and draw detection 
🌐 Flask-based backend API with CORS support 
📊 Clean game logic and modular architecture Tech Stack Python Flask Flask-CORS REST API Use Case



The project demonstrates the implementation of game logic, AI-based decision making, backend development, and API integration in a classic board game environment.
